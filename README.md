# Расширенный метод TOPSIS для группового принятия решений

[![Лицензия: MIT](https://img.shields.io/badge/Лицензия-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![Python 3.8+](https://img.shields.io/badge/python-3.8+-blue.svg)](https://www.python.org/)

Реализация расширенного метода TOPSIS для группового принятия решений с применением к задаче выбора оптимального датасета для SQL-to-text преобразования.

## О проекте
Этот репозиторий содержит Python-реализацию расширенного метода TOPSIS (eTOPSIS) для задач группового принятия решений. Метод позволяет:
- Объективно определять веса экспертов на основе согласованности их оценок
- Агрегировать мнения нескольких экспертов в единое групповое решение
- Ранжировать альтернативы с учетом компетентности каждого эксперта
- Находить компромиссные решения при противоречивых оценках

**Практическое применение рассматривается на конкретной задаче:** Выбор оптимального датасета для задачи преобразования SQL-запросов в текстовое описание (SQL-to-text).

1. **Реализацию на Python** расширенного метода TOPSIS (eTOPSIS)
2. **Практический пример** выбора оптимального датасета для задачи преобразования SQL-запросов в текстовое описание

Метод основан на статье: Yue Z. "A method for group decision-making based on determining weights of decision makers using TOPSIS" (2011).
